# MonAn Ban Chay Nhat API Specification

## Basic Information

* **API Name:** Get Best-Selling Food Items
* **Description:** Retrieves a list of the top K food items with the highest revenue for the currently logged-in shop.
* **HTTP Method:** GET
* **URL Path:** `/api/shop/food/bestSeller`
* **Example URL Request:** `/api/shop/food/bestSeller?k=3`

## Authentication

* **Login Required:** Yes
* **Role:** `Cửa hàng` (Shop Manager)
* **Session Requirement:** Requires `user` (TaiKhoan) and `shopId` (Integer) to be stored in the session.

## Request Parameters

### Path Parameters
None.

### Query Parameters

| Name | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `k` | int | No | The number of top items to retrieve. Defaults to 5 if omitted or invalid. |

### Request Body
None.

## Response

### Success Response

**HTTP Status:** 200 OK

**Example Response:**
```json
[
  {
    "idMonAn": 101,
    "tenMon": "Phở Bò Đặc Biệt",
    "totalRevenue": 2500000.0
  },
  {
    "idMonAn": 105,
    "tenMon": "Cơm Tấm Sườn Bì",
    "totalRevenue": 1850000.0
  }
]
```

| Field | Type | Description |
| ----- | ---- | ----------- |
| `idMonAn` | int | Unique identifier of the food item. |
| `tenMon` | String | Name of the food item. |
| `totalRevenue` | double | Total revenue generated by this food item (Quantity * Price). |

### Error Responses

#### 401 Unauthorized
Returned when the user is not logged in as a Shop Manager.

**HTTP Status:** 401 Unauthorized
```json
{
  "success": false,
  "message": "Unauthorized: Please log in as a Shop Manager."
}
```

#### 500 Internal Server Error
Returned when there is a database error or data retrieval failure.

**HTTP Status:** 500 Internal Server Error
```json
{
  "success": false,
  "message": "Failed to retrieve best-selling items."
}
```

## Business Rules

* Only sales from successfully delivered orders are counted towards revenue (handled in database procedure).
* The `shopId` is securely retrieved from the server-side session.

## Global API Conventions

### Content Type
`application/json`

### Character Encoding
`UTF-8`

### Session Handling
`Cookie: JSESSIONID`
